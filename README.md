# 专属礼物网页

送给朋友的专属网页，包含秋千动画和肉松美食展示。

#### 介绍

一个温馨的专属空间，需要输入密码才能进入。

#### 功能

- 密码登录验证

- 认识天数计算

- 3D秋千动画

- 肉松美食图片轮播

#### 文件说明

- `index.html` - 登录页面

- `question.html` - 天数验证页面

- `gift.html` - 礼物展示页面

- `Images/` - 图片资源文件夹

#### 使用说明

1. 访问网站首页

2. 输入密码（我们认识的日期）

3. 回答认识天数问题

4. 进入专属乐园

#### 部署

直接通过 GitHub Pages 部署即可访问。
